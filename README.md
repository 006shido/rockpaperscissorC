# 🎮 Rock Paper Scissors in C

A simple console-based Rock Paper Scissors game built using the C programming language.

## ✨ Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
- 🎯 Best of 3 and Best of 5 modes
- 📊 Live score tracking
- 🔄 Play again option
- ✅ Input validation
- 🤖 Random computer opponent
- 🏆 Match winner announcement
- 📅 Round counter display


## 🛠 Technologies Used

- C Programming Language
- Standard C Libraries
  - stdio.h
  - stdlib.h
  - time.h
